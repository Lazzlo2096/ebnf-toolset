HOW TO INSTALL
 1) copy the distributed tree to any location
 2) see ebnf.bash or ebnf.cshrc files for required settings
 3) run the tools (e.g., ebnf2annot.pl data/ebnf.txt > /tmp/ebnf.html)
 4) open a html browser with that generated file
